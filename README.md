Introduction:This is a blog platform build using Django , for Users to Create and Manage Posts,read articles and Comment on the Content:
It allows user to search blog posts by title,category and author.used django built authentication for secure login and Registration .Role based accessAuthors manage their posts and readers manage their comments.
